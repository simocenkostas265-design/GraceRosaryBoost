# GraceRosaryBoost - GrRB
Automates the use of Rosary Ring abilities in Benediction to quickly change crouch speed. No more manual button presses. Mapped to Mouse4/5 (choose one of the two), full keyboard customization coming soon. Disable the status indicator in the GUI. Optimized for the Sorrow animation; more animation options will be added in future updates. A lightweight, ergonomic, and practical tool.
